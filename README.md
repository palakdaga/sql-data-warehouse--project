# 🏗️ SQL Data Warehouse Project

Building a modern **Data Warehouse using SQL Server**, including ETL pipelines, data modeling, and analytics.

---

## 🚀 Overview

This project demonstrates an end-to-end **data warehousing solution**, covering:

- Data ingestion from multiple sources (ERP & CRM)
- Data transformation using ETL pipelines
- Data modeling using Star Schema
- Analytics & reporting using SQL

It is designed as a **portfolio project** to showcase real-world data engineering and analytics skills.

---

## 🧱 Data Architecture

This project follows the **Medallion Architecture**:

### 🟤 Bronze Layer
- Raw data ingestion from CSV files
- Stored as-is in SQL Server
- No transformations applied

### ⚪ Silver Layer
- Data cleaning and validation
- Standardization and normalization
- Preparing structured data

### 🟡 Gold Layer
- Business-ready data
- Star schema modeling (Fact & Dimension tables)
- Optimized for analytics and reporting

---

## 🔄 ETL Pipeline

- Extract data from ERP & CRM CSV files  
- Transform data (clean, join, normalize)  
- Load into SQL Server (Bronze → Silver → Gold)

---

## 📊 Data Modeling

- Fact Tables (e.g., Sales)
- Dimension Tables (Customer, Product, Time)
- Designed for fast analytical queries

---

## 📈 Analytics & Reporting

SQL queries are used to generate insights on:

- Customer Behavior
- Product Performance
- Sales Trends
